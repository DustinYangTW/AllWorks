【功能介紹】</br>
透過原生的JavaScript做出相對應的功能，裡面有</br>
1.撥放音樂</br>
2.暫停</br>
3.上一首、下一首</br>
4.快轉5秒、倒轉5秒</br>
5.靜音</br>
6.循環播放</br>
7.隨機播放</br>
8.全曲循環</br>
9.音樂資料庫</br>
